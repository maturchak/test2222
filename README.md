## Задание

Дан массив действительных чисел a₀, ..., aₙ₋₁. 

**Вычислить:**
1. Сумму положительных элементов данного массива
2. Произведение элементов с четными индексами данного массива

**Входные данные:** массив действительных чисел, длина массива может быть любой

**Выходные данные:** 
- Сумма положительных элементов
- Произведение элементов с четными индексами
